## Installation

* Install onemetric from PyPI (recommended):

   ```console
   pip install onemetric
   ```
  
* Install onemetric from the GitHub source:

   ```console
   git clone https://github.com/SkalskiP/onemetric.git
   cd onemetric
   python setup.py install
   ```
